1. 运行依赖

1.1 安装vagrant

1) 确认您的操作系统. 通过 vagrant 官网的 下载地址 下载对应的 vagrant 安装包并安装.

2) 或通过 yum , apt-get 等软件包管理器自行安装.

3) 使用 vagrant --help 确认 vagrant 是否安装成功.


1.2 安装VirtualBox

1) 通过 VirtualBox 官网的 下载地址 下载对应的 VirtualBox 安装包并安装.

2) 或通过 yum , apt-get 等软件包管理器自行安装.

3) 使用 virtualbox --help 确认 VirtualBox 是否安装成功.

2. 导入操作系统

2.1 下载系统镜像

1) 运行虚拟机需要指定所需的操作系统, 通过 vagrant 官方的 镜像搜索 工具搜索想要的操作系统镜像并下载.

2) 快捷下载地址:   Ubuntu 18.04   Ubuntu 20.04   CentOS 7

3) 请不要使用官网提供的镜像使用方法创建虚拟机, 除非您的计算机能快速访问海外网络.
